Created by Arvin Noer Hakim
(c) 2024


------ Persiapan ------
1. Pastikan laptop/pc sudah terinstal Xampp (untuk localhost)

------ Langkah Instalasi ------
1. Extract zip
2. Copy folder Elbayan ke folder .../xampp/htdocs/(disini)
3. Buka xampp
4. Start Apache dan mysql
5. Buka Browser
6. Ketikkan localhost/Elbayan
7. Enjoy.

------ Catatan Rilis ------
# website ini dibuat dinamis, menggunakan template footer, header, post, dan index.php (homepage)
# halaman post page menggunakan post-template, untuk membuat post tinggal mengikuti contoh sepert boga.php, tinggal memasukkan judul artikel, gambar, dan isi artikel.
# belum mobile friendly, pengembangan awal baru fokus di tampilan desktop. Kedepannya akan dibuat agar responsive di mobile.
